🌟 CityStyle Footwear Management System 🌟

CityStyle Footwear is a boutique retail automation system designed to streamline sales and inventory tracking for a modern shoe boutique. This system replaces manual processes with a robust digital solution to manage a diverse range of footwear, from performance sneakers to formal loafers.

🚀 Key Features
📍 Role-Based Access Control: Secure login system with distinct dashboards for Managers and Cashiers.
📍 Inventory Management (CRUD): Full control over shoe records, including adding, updating, and deleting stock with dynamic category loading.
📍 Supplier Management: Centralized database to manage footwear vendors and contact details.
📍 Pricing & Promotions: Bulk price adjustment tools for seasonal discounts and increases.
📍 Automated Reporting: Generates instant inventory reports into text files for administrative review.
📍 Low Stock Alerts: Automated system notifications when shoe items reach critical stock levels.

🛠 Tech Stack
💻 Frontend: Java Swing (GUI)
⚙️ Backend: Java JDK 17+
🗄️ Database: PostgreSQL
🧩 Architecture: Object-Oriented Programming (OOP) principles (Inheritance, Encapsulation, Polymorphism, Abstraction)

📂 Project Structure
📁 src/ : Contains all Java source files (Entity classes, Controller logic, and GUI forms).
📁 lib/ : External libraries and JDBC drivers.
📁 database/ : SQL scripts to set up the PostgreSQL database schema.

⚙️ How to Run
1️⃣ Clone the repository to your local machine.
2️⃣ Set up the PostgreSQL database using the provided script in the /database folder.
3️⃣ Configure the DatabaseConnection.java file with your database credentials.
4️⃣ Run the Login.java file to start the application.
